第 1 題

配分: 15分

請解釋何謂 Context Switch？並詳細說明在作業系統中，發生 Context Switch 的時機以及其對系統帶來的效能影響（Overhead）。

討論區 (3)

答
助教 (TA_Alan) 📌 正確答案 2024-05-11 09:00

官方最佳解：Context Switch 的主要 Overhead 包含：保存與載入 CPU 暫存器狀態、更新 PCB、切換記憶體分頁表（Page Table），以及最嚴重的快取記憶體失效（Cache Miss / TLB Flush）。在撰寫時若能點出 Cache 失效，這題就能拿滿分。

沒錯！還要記得寫到 Context switch time 是一種 pure overhead，因為這段時間系統做不了其他有用的事情，只能做狀態切換。
