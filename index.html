AWS Serverless Exam System

線上考古題管理與討論系統

學生登入後只能存取自己課程的考古題；上傳圖片或手寫答案後，系統會送到 S3， 再由 Lambda 觸發 OCR 流程，最後讓使用者確認辨識文字是否正確。

1. 登入保存 user_id、role、course access。
2. 上傳選擇課程與檔案類型，送到 S3。
3. OCRLambda + Rekognition 處理英文圖片。
4. 確認前端顯示 OCR 結果，使用者可修正。

Demo 預設帳號：student1@nthu.edu.tw、student2@nthu.edu.tw、teacher@nthu.edu.tw。密碼任意。
